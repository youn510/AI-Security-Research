## 3. Key Mechanisms (핵심 공격/방어 원리)

### 3.1. 인지 무선통신의 핵심 기술
- **원리:** CRN에서는 SU가 통신 요구에 따라 스펙트럼 접근을 요청, 융합 센터(FC)는 대역 가용성과 규제 정책에 따라 이러한 요청을 처리한 후, 제어 채널을 통해 사용자에게 적절한 주파수 대역을 할당하여 지정된 스펙트럼 내에서 무선 통신을 가능하게 함.

### 3.2. PUEA (Primary User Emulation Attack)
- **PUEA:** 고전적인 데이터 채널 공격이자 주요 물리 계층 위협으로, CRN에서 가장 초기에 발견되고 가장 광범위하게 연구된 취약점 중 하나. 
- **공격 분류:**
  - **이기적 공격:** 자신의 대역폭 활용을 극대화하고 다른 SU의 통신을 방해하려는 의도.
  - **악의적 공격:** PU를 모방한 신호의 지속적이고 방해적인 전송을 수반하며, 종종 채널 점유를 유지하고 전체 네트워크 성능을 저하시키기 위해 라운드 로빈 패킷 전송 전략을 사용.
- **해결책:** 네트워크 내 악의적 행위자의 신뢰도를 낮추고, 필요 시 운영 무결성을 유지하기 위해 배제를 강제하기 위한 효과적인 대응책이 구현되야함.

### 3.3. SSDF (Spectrum Sensing Data Falsification Attack)
- **SSDF:** 하나 이상의 MU가 의도적으로 위조된 스펙트럼 감지 데이터를 FC에 보고하여 특정 주파수 대역에서 PU의 실제 활동 상태에 대해 대응할 때 발생.
- **피해:** FC가 잘못된 스펙트럼 할당 결정을 내리게 하여 PU에 유해한 간섭을 유발하고 스펙트럼 활용 효율을 현저히 저하시킬 수 있음.
- **공격 분류:**
  - **All-0**
  - **All-1**
  - **역방향 탐지 공격**
  - **동적 공격**

### 3.4. Jamming Attack
- **공격 원리:** 외부 기관(일반적으로 재머라 함)이 합법적인 사용자 간 통신을 방해하기 위해 의도적으로 수행하는 무선 주파수 간섭 형태.
- **공격 방식:** 재머는 고출력 신호를 송출하여 전송을 방해하거나 압도함으로써 합법적 사용자가 상당한 대역폭과 계산 자원을 소모하는 악성 패킷을 수신하도록 강요.
- **피해:** 서비스 거부(DoS)를 초래하여 네트워크 가용성과 성능을 심각하게 저하.
- **공격 분류**
  - **지속적 재밍**
  - **스푸핑 재밍**
  - **무작위 재밍**
  - **반응형 재밍**
 
### 3.5. CUEA (Cognitive User Emulation Attack)
- **공격 방식:** 주파수 핸드오프 단계에서 MU는 핸드오프 인지 사용자(HCU)를 대상으로 CUEA로 알려진 표적 공격을 수행
- **공격 원리:** MU는 주파수 전환 과정에 내재된 지연을 악용하여 합법적인 인지 사용자를 모방함으로써 사용 가능한 채널을 선점하거나 점유하여 HCU가 미사용 주파수 자원에 접근하지 못하게 함.
- **피해:** 정상적인 데이터 전송을 방해하고 전체 네트워크 성능을 크게 저하시킴.

### 3.6. Objective Function Attack
- **공격 방식:** 목적 함수 공격에 취약하며, 이때 다중 사용자(MU)는 단일 사용자(SU)의 무선 매개변수 최적화를 전복시킴.

### 3.7. Eavesdropping Attack 
- **공격 방식:** 무선 주파수(RF) 및 방송 무선 통신 시스템에서, MU는 CRN 내에서 전송되는 기밀 정보를 가로채고 추출하기 위해 도청 공격을 수행함.
- **공격 분류:**
  - **수동적 도청:** 통신 과정을 변경하지 않고 전송된 데이터를 조용히 감시 및 수집하는 방식.
  - **능동적 도청:** 통신을 가로채는 것뿐만 아니라 이후 통신을 조작하는 행위까지 포함.

## 4. My Insight (나의 생각 & 적용점)
- **차이점:** 기존 인터넷 해킹은 '데이터'를 조작하지만, 무선 보안은 '신호(Signal)와 주파수'자체를 무기로 쓴다는 점이 신기했다.
- **적용점:** 앞으로 IoT나 드론 보안을 공부할 때, 이런 무선 신호 기반의 공격을 막는 기술이 필수적이라고 생각한다. 이러한 공격들을 막기 위한 공부들을 더 해야겠다.
